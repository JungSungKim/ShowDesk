### 세션 2 — Phase 1 전체 완료

**3D 뷰어 핵심**
- `src/core/loader/stlLoader.ts` — ASCII/Binary STL 파싱
- `src/core/renderer/sceneManager.ts` — Three.js 씬, OrbitControls, 조명
- `src/renderer/components/Viewer3D.tsx` — React 캔버스 컴포넌트
- 렌더링 모드: Shaded / Wireframe(EdgesGeometry 5°, #39FF14) / Shaded+Edge
- 스테이지 조명: SpotLight(decay=0) + 그림자 + 어두운 배경(#16162a)
- 와이어프레임 후면 엣지: `colorWrite=false, depthWrite=false` 트릭 적용
- Ghost 모드(G) + Isolate 모드(I) 구현

**BOM 연동**
- `src/core/bom/bomParser.ts` — CSV 컬럼 자동 감지, 트리 구성
- `src/core/bom/xlsxLoader.ts` — SheetJS로 xlsx → csv 변환
- `src/renderer/components/PartTree.tsx` — 계층 트리 UI (검색 필터 포함)
- BOM-first 워크플로우, 파트별 STL 수동 지정
- Auto Map: 폴더 선택 → 파일명 일치하는 STL 일괄 매핑

**UI/UX**
- HUD 오버레이 레이아웃: 파트 트리·인포 패널이 3D 뷰 위에 부유
- 파트 정보 패널: 게임 스테이터스 스타일
- 랜딩 스크린: 3가지 진입 경로 카드
- 뷰 큐브(1/2/3), Fit(F), 스크린샷(Ctrl+Shift+S → PNG)

**프로젝트 파일**
- `.showdesk` 포맷: BOM 경로 + 파트별 STL 경로 JSON 저장/복원
- IPC 채널 추가: `dialog:openDirectory`, `fs:readDir`
