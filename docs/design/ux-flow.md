## 1. 전체 화면 레이아웃

```
┌─────────────────────────────────────────────────────────────┐
│  [툴바] 파일열기 | 렌더모드 | Ghost | Isolate | Fit | 스크린샷  │
├──────────────┬──────────────────────────────┬───────────────┤
│              │                              │               │
│  파트 트리   │       3D 뷰어                │  파트 정보    │
│  (BOM 계층)  │                              │  패널         │
│              │      [Three.js Canvas]       │               │
│  - Assembly  │                              │  파트 번호:   │
│    ├ Part A  │                              │  파트 이름:   │
│    ├ Part B  │                              │  수량:        │
│    └ Part C  │                              │  재질:        │
│              │                              │               │
│              │                              │               │
├──────────────┴──────────────────────────────┴───────────────┤
│  [상태바] 로드된 파일명 | 파트 수 | 선택된 파트명             │
└─────────────────────────────────────────────────────────────┘
```

### 패널 비율 (기본)
- 파트 트리: 20%
- 3D 뷰어: 60%
- 파트 정보: 20%
- 각 패널 너비 드래그로 조절 가능

## 2. 주요 사용자 흐름

### 흐름 A — STL만 열어 탐색

```
앱 실행
  └→ 빈 뷰어 화면 (드래그&드롭 안내 메시지 표시)
       └→ STL 파일 드래그&드롭 또는 Ctrl+O
            └→ 로딩 프로그레스바
                 └→ 3D 모델 렌더링 + 자동 Fit
                      └→ 마우스로 회전/줌/팬 탐색
```

### 흐름 B — BOM 연동 리뷰

```
앱 실행
  └→ STL 파일 로드 (흐름 A)
       └→ BOM 파일 로드 버튼 클릭
            └→ CSV/Excel 파일 선택
                 └→ 파트 트리 자동 구성
                      ├→ 매핑 성공: 파트 트리에 정상 아이콘
                      └→ 매핑 실패: 경고 아이콘 표시
                           └→ (선택) 수동 매핑 UI 진입
                                └→ 리뷰 시작
                                     └→ 파트 트리 클릭
                                          ├→ 3D 하이라이트
                                          └→ 파트 정보 패널 업데이트
```

### 흐름 C — Ghost/Isolate 집중 뷰

```
파트 선택 (트리 또는 3D 직접 클릭)
  ├→ G 키 → Ghost 모드: 나머지 반투명
  ├→ I 키 → Isolate 모드: 나머지 숨김
  └→ ESC → 전체 초기화
```

## 3. 파트 트리 UI 상세

```
파트 트리 패널
┌─────────────────────────┐
│ 🔍 검색...              │
├─────────────────────────┤
│ ▼ 📦 ASSEMBLY_TOP       │
│   ▼ 📦 SUB_ASSY_01      │
│     ├ ✅ PART_001       │  ← 매핑 성공
│     ├ ✅ PART_002       │
│     └ ⚠️  PART_003      │  ← 매핑 실패
│   ├ ✅ PART_010         │
│   └ ✅ PART_011         │
└─────────────────────────┘
```

- 검색창: 파트 번호 또는 이름으로 필터
- 우클릭 메뉴: [하이라이트] [Ghost] [Isolate] [정보 보기]

---

## 4. 매핑 실패 처리 UX

```
⚠️ 매핑 실패 파트 클릭
  └→ 하단 토스트: "PART_003 — 3D 오브젝트를 찾을 수 없습니다. 수동 연결하시겠습니까?"
       └→ [연결하기] 버튼 클릭
            └→ 3D 뷰어에서 오브젝트 클릭 모드 활성화
                 └→ 원하는 3D 오브젝트 클릭
                      └→ 매핑 저장 → ✅ 아이콘으로 변경
```

---

## 5. 뷰 큐브 (View Cube)

```
뷰어 우측 상단 고정
     [TOP]
[LEFT][FRONT][RIGHT]
     [BOTTOM]
```
- 각 면 클릭 시 해당 방향 뷰로 애니메이션 전환
- 드래그로 뷰 큐브 자체를 회전하면 3D 모델도 동기화

---

## 6. 온보딩 (첫 실행)

```
첫 실행 시
  └→ 툴팁 오버레이 표시 (3단계)
       1단계: "STL 파일을 여기에 드래그하세요"
       2단계: "BOM 파일을 불러오면 파트 트리가 생성됩니다"
       3단계: "파트 트리에서 파트를 클릭해 하이라이트하세요"
       └→ [시작하기] 버튼으로 온보딩 완료
```
