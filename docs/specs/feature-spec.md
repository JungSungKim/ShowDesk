## F-01. STL 파일 로드

### 설명
사용자가 하나 또는 여러 개의 STL 파일을 불러와 3D 뷰어에 표시한다.

### 동작 방식
- 파일 탐색기 열기 버튼 또는 드래그&드롭으로 파일 선택
- 다중 STL 로드 시 각 파일을 별도 오브젝트로 씬에 추가
- 로드 진행률 프로그레스바 표시
- 로드 완료 후 모델 전체가 화면에 맞게 자동 Fit

### 입력
- 파일 형식: `.stl` (ASCII / Binary 모두 지원)
- 최대 파일 크기: 200MB (단일 파일)

### 출력
- 3D 뷰어에 메시 렌더링
- 왼쪽 파일 트리 패널에 파일명 목록 추가

### 예외 처리
- 지원하지 않는 포맷 → 오류 토스트 메시지
- 파일 손상 → 파싱 오류 메시지 및 무시 후 계속

## F-02. 3D 뷰어 조작

### 설명
마우스로 모델을 자유롭게 탐색한다.

### 조작 방식

| 동작 | 입력 |
|---|---|
| 회전 | 마우스 왼쪽 버튼 드래그 |
| 이동(팬) | 마우스 가운데 버튼 드래그 또는 Shift + 왼쪽 드래그 |
| 줌 | 마우스 휠 스크롤 |
| 전체 Fit | F 키 또는 툴바 버튼 |
| 정면/측면/상면 뷰 | 뷰 큐브 클릭 또는 단축키 (1/2/3) |

### 렌더링 모드

| 모드 | 설명 |
|---|---|
| Shaded | 기본 음영 렌더링 |
| Wireframe | 와이어프레임 표시 |
| Shaded + Edge | 음영 + 엣지 라인 |

## F-03. BOM 파일 로드 및 파트 트리

### 설명
BOM 파일을 불러와 계층 구조 트리로 표시하고 3D 모델과 연동한다.

### 지원 형식
- CSV (UTF-8, 쉼표 구분)
- Excel (.xlsx)

### BOM 필수 컬럼

| 컬럼명 | 설명 | 필수 여부 |
|---|---|---|
| part_number | 파트 번호 (3D 오브젝트명과 매핑 키) | 필수 |
| part_name | 파트 이름 | 필수 |
| quantity | 수량 | 선택 |
| material | 재질 | 선택 |
| parent_part | 상위 파트 번호 (계층 구조용) | 선택 |

### 동작 방식
- BOM 로드 시 `part_number` 기준으로 STL 오브젝트명과 자동 매핑
- 매핑 실패 항목은 파트 트리에 경고 아이콘으로 표시
- 수동 매핑 UI: 미매핑 파트 우클릭 → 3D 오브젝트 연결

## F-04. 파트 하이라이트

### 설명
BOM 트리에서 파트를 선택하면 3D 뷰어에서 해당 파트가 강조 표시된다.

### 동작 방식
- 파트 트리 항목 클릭 → 해당 3D 오브젝트 하이라이트 (기본 색상: 주황)
- 3D 뷰어에서 오브젝트 직접 클릭 → 파트 트리에서 해당 항목 자동 선택 및 스크롤
- 다중 선택: Ctrl + 클릭

### 하이라이트 색상
- 선택: `#FF6B00` (주황)
- 다중 선택 추가: `#FFD700` (골드)
- 기본 상태: `#AAAAAA` (회색)

## F-05. Ghost 모드

### 설명
선택한 파트 외의 나머지 파트를 반투명하게 처리하여 선택 파트에 집중하게 한다.

### 동작 방식
- 파트 선택 후 Ghost 버튼 클릭 또는 단축키 `G`
- 비선택 파트 투명도: 15%
- Ghost 해제: 동일 단축키 또는 버튼 재클릭, 빈 공간 클릭

---

## F-06. Isolate 모드

### 설명
선택한 파트만 화면에 표시하고 나머지는 숨긴다.

### 동작 방식
- 파트 선택 후 Isolate 버튼 클릭 또는 단축키 `I`
- 비선택 파트 완전 숨김 (visibility = false)
- Isolate 해제: ESC 키 또는 버튼 재클릭

---

## F-07. 파트 정보 패널

### 설명
선택한 파트의 BOM 정보를 오른쪽 패널에 표시한다.

### 표시 항목
- 파트 번호
- 파트 이름
- 수량
- 재질
- 기타 BOM 커스텀 컬럼

---

## F-08. 스크린샷 저장

### 설명
현재 3D 뷰어 화면을 이미지 파일로 저장한다.

### 동작 방식
- 툴바 카메라 버튼 클릭 또는 단축키 `Ctrl+Shift+S`
- 저장 형식: PNG
- 저장 경로: 사용자 지정 또는 기본 Documents 폴더
- 배경 포함/제외 옵션
