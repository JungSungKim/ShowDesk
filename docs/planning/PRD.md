# ShowDesk — Product Requirements Document

**버전:** 0.1  
**작성일:** 2026-05-27  
**상태:** 초안

---

## 1. 제품 개요

### 1.1 제품 정의
ShowDesk는 3D CAD 설계 파일(STL)과 BOM(Bill of Materials)을 연동하여, 설계 전문가가 아닌 일반 참여자도 회의 자리에서 3D 모델을 직관적으로 이해하고 탐색할 수 있도록 돕는 PC 데스크탑 기반 쇼잉(Showing) 툴이다.

### 1.2 해결하는 문제
- 기존 설계 리뷰 회의에서는 CAD 소프트웨어를 다룰 수 있는 사람이 없으면 스크린샷 이미지에 의존
- 이미지는 시점이 고정되어 있어 내부 구조, 조립 관계, 파트 위치를 직관적으로 파악하기 어려움
- BOM과 3D 모델을 동시에 참조하려면 두 개의 도구를 번갈아 사용해야 하는 불편함

### 1.3 타겟 사용자

| 사용자 유형 | 설명 |
|---|---|
| 주 사용자 | 설계 담당자 (회의 진행자, 뷰어 조작) |
| 보조 사용자 | 비전문가 참여자 (고객, 생산팀, 경영진) |

### 1.4 사용 환경
- **플랫폼:** Windows PC (데스크탑/노트북)
- **사용 맥락:** 회의실 프로젝터 또는 모니터 공유
- **네트워크:** 오프라인 사용 가능 (로컬 파일 기반)

## 2. 목표 및 성공 지표

### 2.1 제품 목표
1. STL 파일을 별도 CAD 소프트웨어 없이 즉시 열고 3D로 탐색할 수 있다
2. BOM 파일을 불러오면 파트 트리가 자동 구성되고 3D 모델과 연동된다
3. 비전문가가 5분 이내에 기본 조작을 익힐 수 있다

### 2.2 성공 지표 (MVP 기준)
- STL 로드 시간: 50MB 파일 기준 5초 이내
- BOM-3D 매핑 성공률: 올바른 네이밍 규칙 적용 시 90% 이상
- 사용자 학습 시간: 핵심 기능 5분 이내 숙지

---

## 3. 범위

### 3.1 In Scope (MVP)
- STL 파일 단일/다중 로드
- 기본 3D 뷰어 (회전, 줌, 팬)
- CSV/Excel BOM 파일 파싱 및 파트 트리 표시
- BOM 파트 선택 시 3D 하이라이트
- Ghost 모드 (선택 파트 외 반투명 처리)
- Isolate 모드 (선택 파트만 단독 표시)
- 스크린샷 저장

### 3.2 Out of Scope (MVP 이후)
- STEP / IGS / WRL 포맷 지원
- 어노테이션 핀 삽입 및 저장
- Exploded View 자동 생성
- 단면도 슬라이싱
- 원격 공유 세션
- 버전 비교

---

## 4. 제약 조건
- 오픈소스 라이브러리만 사용 (상업 라이선스 비용 최소화)
- 인터넷 연결 없이 동작해야 함
- 설치 패키지 크기 200MB 이내 목표
